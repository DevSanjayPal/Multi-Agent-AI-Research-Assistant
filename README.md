# 🤖 Multi-Agent AI Research Assistant

A Multi-Agent AI Research Assistant that automates the complete research workflow using Large Language Models (LLMs). The system employs specialized AI agents to search the web, extract relevant information, generate structured research reports, and critically evaluate the generated reports.

## Features

- 🔍 AI-powered web search using Tavily Search API
- 🤖 Multi-Agent architecture with LangChain
- 📖 Intelligent website content scraping using BeautifulSoup
- ✍️ Automated research report generation
- 🧐 AI-based report evaluation and scoring
- ⚡ LCEL (LangChain Expression Language) pipeline implementation
- 💬 Prompt Engineering for structured outputs
- 🌐 Interactive Streamlit web interface
- 🔒 Secure API key management using environment variables
